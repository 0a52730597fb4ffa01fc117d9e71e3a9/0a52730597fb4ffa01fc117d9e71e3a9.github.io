Docker镜像下载与文件提取

2023-02-04 / 1 min read

有时候遇到pull下来镜像，无法正常启动的情况。这时候需要提取镜像中的文件进行分析，这里简单做个笔记，避免后面忘记。

0x01 添加registries

在Docker Engine中添加insecure-registries或registry-mirrors.

"insecure-registries": ["127.0.0.1:80"],
"registry-mirrors": [ "http://127.0.0.1:80"]

0x02 registries认证

docker login http://127.0.0.1:80

0x03 镜像下载与提取

\# 通过pull下载registries中的镜像
docker pull demo:latest
\# 通过save保存registries中的镜像为压缩文件
docker save -o demo.tar demo:latest
